# Product Purchase Service

Тестовое Spring Boot приложение для расчета цены продукта и проведения оплаты.

Проект будет собран вокруг двух сценариев:

- расчет итоговой цены с учетом купона и налога;
- проведение покупки через выбранный платежный процессор.

Документацию и детали запуска дополню после реализации основного функционала.
